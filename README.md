# Directorio Académico UdeC

Sistema de consola desarrollado en Dart para administrar registros de alumnos y maestros de la Universidad de Colima. El proyecto implementa operaciones básicas de tipo CRUD para crear, consultar, actualizar y eliminar registros dentro de un directorio académico.

## Descripción

Este proyecto fue desarrollado como práctica final para aplicar conceptos fundamentales de programación orientada a objetos en Dart. La solución utiliza herencia, listas, entrada por consola y lógica de control para gestionar dos tipos de registros:

- alumnos
- maestros

Cada registro almacena información básica como:

- identificador
- nombre
- apellido paterno
- apellido materno
- correo electrónico

## Funcionalidades

- crear nuevos registros
- listar todos los registros
- filtrar registros por tipo
- actualizar información existente
- eliminar registros por ID

## Cómo ejecutar

1. Instala el SDK de Dart.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el archivo principal con:

```bash
dart run directorio_academico_udec.dart
```

O bien:

```bash
dart directorio_academico_udec.dart
```
